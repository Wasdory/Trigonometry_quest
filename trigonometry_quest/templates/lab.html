Справка по вводу функций

  • Основные функции: sin(x), cos(x), tan(x)
  • Математические операции: +, -, *, /
  • Число π: используйте pi или π
  • Степень: используйте ** (например: x**2)
  • Скобки: для группировки используйте ( )
  • Константы: можно использовать числа, pi, e
